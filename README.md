# Docker setup
This system is a bundle created to provide services related to jitsi video metting
## System features
- Jitsi videocall service
- OpenVpn dashboard
- OpenVpn service
- Traefik as a reverse proxy
- Traefik dashboard
- Calendar service to schedule videocalls
## System requirements
- Unix system
- Docker and docker-compose installed
- 4 domains to vpn dashboard,jitsi,traefik dashboard and calendar
- Server with open ports on firewall (at least 80,443 and 1194)
- Andministrator email
